Projet par défaut d'iGRC Analytics

Cette facette contient les données du projet par défaut.

  • Détail
  • Notes d'utilisation
  • Historique des versions

Détail

Cette facette ne peut être installée ou désinstallée comme les autres facettes ; elle contient l'ensemble des données du projet par défaut lors de sa création. Elle peut servir à comparer les fichiers modifiés par rapport à leur version originale, à la réparer ou encore à migrer la totalité du projet par défaut dans le cas où une mise à jour des fichiers est disponible.

Notes d'utilisation

Cette facette est installée automatiquement lorsque l'on démarre un nouveau projet ; aucune action spécifique de configuration ou d'installation n'est nécessaire.
